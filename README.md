# Vasudutt Pareek — Portfolio

A cinematic, scroll-driven developer portfolio. **Bold red. Falling snow. A green football pitch for the tech stack. A spider that rappels as you scroll.** Built with [Astro](https://astro.build), GSAP and Lenis.

🔗 **Live:** _add your Netlify URL here_

---

## ✨ Features

- **Multilingual "Hello" intro** and a **"Goodbye" outro** that cycles through languages.
- **Repaint-the-site colour picker** — change the entire theme to any colour live, with auto-contrasting text (try Noir & Bone).
- **The Pitch** — my tech stack laid out as a football starting XI on a green field; tap any player to scout it.
- **Laurels** — animated awards/credentials grid with linked certificates.
- **Crashed Back to Earth** — a wireframe globe with a rappelling rocket.
- **WORK** — a 3D fly-through showcase of projects.
- **ECHO** — a built-in assistant chatbot.
- **Contact form** with a cinematic video background (EmailJS).
- Fully responsive, with a custom spider favicon.

## 🧰 Tech stack

`Astro` · `TypeScript` · `SCSS` · `GSAP + ScrollTrigger` · `Lenis` · `EmailJS`

## 🚀 Run locally

```bash
npm install
npm run dev      # dev server → http://localhost:4321
npm run build    # production build → ./dist
npm run preview  # preview the production build
```

## ☁️ Deploy (Netlify)

This repo ships a `netlify.toml`, so Netlify auto-detects everything:

- **Build command:** `npm run build`
- **Publish directory:** `dist`

Import the repo on Netlify → Deploy. Every push redeploys automatically.
